# Submissions

Students submit their `decision.json` files here via Pull Request.

## How to submit

1. Fork this repository
2. Create your decision file: `submissions/ep01/YOUR_STUDENT_ID.json`
3. Open a Pull Request to this repository
4. The auto-validator will check your submission

## File format

Each submission must be a valid JSON file following the schema in `episodes/ep01/schema.json`.
